CycleganSA

CycleganSA is a PyTorch-based implementation of CycleGAN that includes a self-attention mechanism and semantic segmentation. It can generate images faster than the original CycleGAN and improves upon the blurred background issue by adding semantic segmentation.

Features of [CycleganSA](https://github.com/Topaz1618/CycleganSA/):

  • Self-attention mechanism
  • Semantic segmentation
  • Faster image generation than CycleGAN

MeowFile

MeowFile is a Python-based file management system that allows users to upload, download, batch manage, and review documents. It is built using the Tornado web framework and uses MySQL as its database. MeowFile also includes features such as user login/logout, password reset, and authority management. It offers a simple installation guide and usage instructions, supporting Ubuntu16.04 and Mac OS.

Features ofMeowFile:

  • User login/logout, password reset, authority management
  • File upload (support breakpoint upload, batch upload)
  • File download
  • Batch management of files
  • Document review process (Only for users with intranet permission)

Plasticine

Plasticine Plasticine is a facial deformation application that uses face recognition technology to detect key points on a face and apply deformations to the cheeks, chin, and cheekbones. It is available on Ubuntu and macOS.

Tech Stack of Plasticine:

  • Python
  • TensorFlow
  • OpenCV
  • numpy

DonutsTerminal

DonutsTerminal is a Python-based web application developed using WebSocket, Tornado, and paramiko that allows users to connect to a remote server via SSH. Users can access DonutsTerminal through their web browser and do not need to download a local terminal to connect to the remote server.

Tech Stack of DonutsTerminal:

  • Python
  • WebSocket
  • Tornado
  • Paramiko library for SSH connection.
  • MySQL database for storing user and file information.

FoxMonitor

FoxMonitor is a distributed monitoring system developed based on Django, with reference to Zabbix and Open-Falcon architecture. It implements various functionalities including front-end, back-end, monitoring plug-ins, drawing, data optimized storage, etc. It supports system and service monitoring, data collection, data storage and optimization, data visualization, monitoring alarm, alarm handling and backstage management. The project is developed using Python programming language, Django Web framework, Redis database, and psutil library for system information retrieval. FoxMonitor runs on Linux and Mac systems, provides detailed installation and configuration instructions, and has good cross-platform compatibility and user-friendliness.

Tech Stack of FoxMonitor:

  • Python
  • Django
  • Redis
  • psutil

Features of FoxMonitor:

  • System and service monitoring
  • Data collection
  • Data storage and optimization
  • Data visualization
  • Monitoring alarm
  • Alarm handling
  • Backstage management
